# Project 3. 편의점 상품 분류 AI 학습 데이터 구축

| 항목 | 내용 |
|---|---|
| 기간 | 2026.04.28 ~ 05.13 |
| 팀 구성 | 5명 |
| 담당 | 상품 이미지 라벨링 |
| 툴 | Roboflow |

---

## 프로젝트 개요

협동로봇 비전 시스템에서 편의점 상품을 자동 분류하기 위한 AI 모델 학습용 데이터셋 구축.
Roboflow를 활용하여 캔 음료 4종의 이미지를 라벨링하였다.

---

## 데이터셋 구성

| 항목 | 내용 |
|---|---|
| 총 이미지 수 | 423장 |
| 클래스 수 | 4종 |
| 라벨링 툴 | Roboflow |

**분류 클래스**

| 클래스 | 상품 |
|---|---|
| can_coke_decaf | 코카콜라 일반 |
| can_coke_zero | 코카콜라 제로 |
| can_pepsi | 펩시 일반 |
| can_pepsi_zero | 펩시 제로 |

---

## 담당 역할

- Roboflow 플랫폼에서 캔 음료 이미지 바운딩 박스 라벨링
- 팀원 5명이 분담하여 총 423장 데이터셋 완성
- 라벨링 데이터는 YOLO 기반 상품 분류 모델 학습에 활용

---

## PE 직무 연계

| 현장 경험 | PE 업무 연결 |
|---|---|
| 이미지 데이터 정밀 라벨링 | 공정 불량 이미지 분류 기준 수립 |
| 클래스별 데이터 분류 작업 | 비전 검사 시스템 데이터 관리 |

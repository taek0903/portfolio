# Project 4. Arduino 스마트 실내 환경 자동 제어 시스템

| 항목 | 내용 |
|---|---|
| 기간 | 2021.11 ~ 2021.12 |
| 담당 | 센서 코드 작성 (C/C++) |
| 장비 | Arduino, 가스 센서, 온도 센서, DC 팬, 열선패드, LCD |

---

## 프로젝트 개요

실내의 가스 농도와 온도를 실시간 계측하여, 기준치를 초과하거나 미달할 경우
환풍구·선풍기·열선패드를 자동으로 제어하는 아두이노 기반 임베디드 시스템.
센서 데이터는 LCD 2개에 실시간 출력하며, LED 인디케이터로 시스템 상태를 시각화하였다.
이 중 센서 계측 및 제어 로직 전반의 아두이노 코드 작성을 담당하였다.

---

## 시연 시나리오

| # | 조건 | 동작 결과 |
|---|---|---|
| 1 | 실내 가스 농도 기준치 초과 | 환풍구(DC 팬) 자동 작동 |
| 2 | 실내 가스 농도 기준치 이하로 복귀 | 환풍구 자동 정지 |
| 3 | 실내 온도 기준치 초과 | 선풍기 자동 작동 |
| 4 | 실내 온도 기준치 미만 | 열선패드 자동 작동 |

---

## 담당 역할 — 센서 코드 작성

### 1. 센서 계측 및 조건 분기 제어

- 가스 센서(MQ 시리즈) 아날로그 출력을 ADC로 읽어 임계값과 비교하는 로직 구현
- 온도 센서 값을 폴링 방식으로 읽어 고온/저온 조건에 따라 액추에이터 ON/OFF 제어
- 가스·온도 두 조건을 독립적으로 처리하여 간섭 없이 동시 제어되도록 코드 구성

### 2. 출력 장치 제어 코드

- 환풍구용 DC 팬: 가스 농도 초과 시 작동 / 정상 복귀 시 즉시 정지
- 선풍기용 DC 팬: 온도 기준치 초과 시 냉각 작동
- 열선패드: 온도 기준치 미만 시 가열 작동

### 3. 실시간 데이터 출력

- LCD 16x2 2개에 가스 농도 수치 및 온도 수치 실시간 출력
- LED 인디케이터로 각 장치의 작동 상태를 직관적으로 표시

---

## 사용 부품

| 부품 | 역할 |
|---|---|
| Arduino | 메인 제어 보드 |
| MQ 시리즈 가스 센서 | 실내 가스 농도 계측 |
| 온도 센서 (NTC / DHT) | 실내 온도 계측 |
| DC 팬 × 2 | 환풍구(가스 제어) + 선풍기(온도 제어) |
| 열선패드 | 저온 시 가열 |
| LCD 16x2 × 2 | 가스 농도·온도 실시간 출력 |
| LED 인디케이터 | 시스템 상태 시각화 |
| 아크릴 인클로저 | 실제 환경 모사, 내부 회로 시각화 |

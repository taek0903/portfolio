# Arduino 스마트 실내 환경 자동 제어 시스템

> 2021.11 ~ 2021.12 | 가스 센서 및 온도 센서를 활용한 실내 환경 모니터링 및 자동 제어 시스템

---

## 개요

실내의 가스 농도와 온도를 실시간으로 계측하고, 기준치에 따라 환풍구·선풍기·열선패드를 자동으로 제어하는 아두이노 기반 임베디드 시스템입니다.
센서 데이터는 LCD 디스플레이로 실시간 확인할 수 있으며, LED 인디케이터로 현재 상태를 시각적으로 표시합니다.

---

## 시연 시나리오

| # | 조건 | 동작 |
|---|---|---|
| 1 | 실내 가스 농도 기준치 초과 | 환풍구(DC 팬) 자동 작동 |
| 2 | 실내 가스 농도 기준치 이하로 복귀 | 환풍구 자동 정지 |
| 3 | 실내 온도 기준치 초과 | 선풍기 자동 작동 |
| 4 | 실내 온도 기준치 미만 | 열선패드 자동 작동 |

---

## 사용 부품

| 부품 | 역할 |
|---|---|
| Arduino (Uno / Mega) | 메인 제어 보드 |
| MQ 시리즈 가스 센서 | 실내 가스 농도 계측 |
| 온도 센서 (NTC / DHT) | 실내 온도 계측 |
| DC 팬 (환풍구용) | 가스 농도 초과 시 환기 |
| DC 팬 (선풍기용) | 온도 초과 시 냉각 |
| 열선패드 | 온도 미달 시 가열 |
| LCD 16x2 (x2) | 가스 농도 및 온도 실시간 출력 |
| LED 인디케이터 | 시스템 상태 시각화 |
| 브레드보드 | 회로 구성 |
| 아크릴 인클로저 | 실제 환경 모사를 위한 밀폐 구조물 |

---

## 구현 내용

- 가스 센서 아날로그 값을 ADC로 읽어 임계값과 비교, 초과 시 팬 PWM 제어
- 온도 센서 값을 폴링 방식으로 읽어 고온/저온 조건에 따라 팬 또는 열선패드 ON/OFF
- 두 개의 LCD에 가스 농도·온도 수치를 실시간으로 출력
- LED 신호로 정상/경고 상태 구분 표시
- 아크릴 투명 인클로저로 내부 회로를 시각화하여 시연 완성도 향상

---

## 핵심 역량

- 센서 계측값 기반 조건 분기 및 액추에이터 제어 로직 구현
- 다중 센서·출력 장치의 동시 구동 및 타이밍 제어
- 하드웨어 회로 설계 및 인클로저 직접 제작
